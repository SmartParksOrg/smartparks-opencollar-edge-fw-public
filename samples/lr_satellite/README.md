# LR11xx satellite send example

## Description

Long Range-Frequency Hopping Spread Spectrum (LR-FHSS) is the new physical layer designed to address
extremely long-range and large-scale communication scenarios, such as satellite IoT.

## Sample definition

This sample shows how to initialize LoRa FHSS mode, generate and build a tx_buffer and send said
buffer.

## Developer note

If you're having issues with the code not performing frequency hopping even though enabled, check
the payload length.
