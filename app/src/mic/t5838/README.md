# T5838 Microphone

This module features support for the T5838 Microphone.

Component datasheet can be found
[by following this link](https://invensense.wpenginepowered.com/wp-content/uploads/2022/08/DS-000383-T5838-v1.0.pdf).

## Module capabilities

Currently supported functionality is initialization after which the microphone is put in a low power
state.
